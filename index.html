The Pipeline

SIX
STAGE
WORKFLOW

LUMINA's LangGraph orchestration engine executes a deterministic six-node pipeline with concurrent async execution, Semaphore-controlled parallelism, and WebSocket-driven real-time event streaming.

01
🕷
Repository Ingestion
Parallel async GitHub API crawling with Semaphore(10). Extracts file contents, structure, and metadata at scale.
Semaphore(10)
02
🧠
Semantic Indexing
Language-aware chunking, 3072-dim OpenAI embeddings, batched Pinecone upsert for intelligent retrieval.
Pinecone RAG
03
🗺
Architecture Mapping
RAG retrieval of top-5 chunks. GPT-4o structured analysis generates component relationships and data flows.
GPT-4o
04
🔍
Vuln Detection
Two-phase: regex pre-filtering followed by concurrent LLM verification via asyncio.gather for novel patterns.
Dual-Phase
05
⚙️
Fix Synthesis
Per-vulnerability RAG-augmented patch generation. Retrieves similar patterns, produces production-ready fixes.
Semaphore(5)
06
🚀
Pull Request
Fork repo, create security branch, concurrent commits with Semaphore(3), open descriptive PR automatically.
Auto PR
